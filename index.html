Project 1

EmpowerGlow

EmpowerGlow is an e-commerce platform offering a wide range of women’s hygiene products with a seamless, user-friendly shopping experience. It features an intuitive interface for browsing, managing carts, and wishlists, alongside real-time chat for instant customer support. The platform ensures secure user authentication and efficient data management using Firebase, while React.js powers its dynamic and responsive frontend. With Node.js and Socket.IO enabling real-time communication, EmpowerGlow reflects a customer-first approach by blending advanced technology with thoughtful design to deliver convenience, reliability, and efficiency.

Project 2

H-Plus

The Hospital Management System (H-Plus) is a comprehensive web application designed to streamline hospital operations, providing separate views for patients and doctors. Built using JSP, Servlets, JDBC, MySQL, and Tailwind, the system facilitates patient registration, appointment scheduling, admission, discharge, and billing processes. It ensures secure payment tracking with RESTful APIs and robust session management for reliable performance. With a focus on efficiency and user experience, H-Plus optimizes data transactions and integrates secure workflows for seamless hospital management.
